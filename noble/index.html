noble

noble cryptography is a high-security, easily auditable set of contained cryptographic libraries with following features:
  • Zero or minimal dependencies
  • Highly readable TypeScript / JS code
  • PGP-signed releases and transparent NPM builds with provenance
The idea was to: 1) improve knowledge of cryptography 2) improve supply chain security of JS ecosystem 3) provide libraries that could be used for learning, because other libs are too low-level / unfriendly to beginners. Active projects:
  • ciphers (github, npm): cryptographic ciphers, including Salsa20, ChaCha, AES-SIV, and FF1.
  • curves (github, npm): elliptic curve cryptography, including Weierstrass, Edwards, Montgomery curves, pairings, hash-to-curve, poseidon hash, schnorr, secp256k1, ed25519, ed448, p521, bn254, bls12-381 and others.
  • hashes (github, npm): hash functions, MACs & KDFs, including SHA, RIPEMD, BLAKE, HMAC, HKDF, PBKDF & Scrypt
  • 4kb frozen versions of curves with even better auditability and rarer update schedule: secp256k1 (github, npm) and ed25519 (github, npm)

Brief history of Noble

  • June 2019: secp256k1 (commit) and ed25519 (commit) have been released
  • Aug 2019: bls12-381 has been released
  • Apr 2020: Learning fast elliptic-curve cryptography blog post went live
  • Apr 2021: secp256k1 audit by Cure53 has been crowdfunded by community with help of Umbra.cash
  • Oct 2021: hashes have been released, funded by Ethereum Foundation
  • Nov 2021: received a grant from Optimism
  • Jan 2022: hashes audit by Cure53 has been completed, also funded by Ethereum Foundation, with help of Nomic Labs
  • Feb 2022: ed25519 audit by Cure53 has been completed
  • Dec 2022: curves have been released, consolidating secp + ed + bls in one package
  • Feb 2023: curves audit by Trail of Bits has been completed, funded by Ryan Shea
  • Mar 2023: secp256k1 and ed25519 have been rewritten and became 4KB single-feature versions of curves
  • Jun 2023: ciphers have been released
  • Sep 2023: curves have been audited for the second time, by Kudelski Security

scure

noble's non-cryptography sister project: secure, independently audited packages for every use case. Release and audit timeline:

  • Jan 2022: base base64, bech32..., bip32 hdkey, bip39 mnemonics
  • Feb 2023: btc-signer transactions, segwit, taproot, psbt, multisig
  • Sep 2023: starknet stark curve, pedersen and poseidon hash

Demo 1: Elliptic curve calculator

This is a simple form that allows to calculate elliptic curve public keys and signatures with noble. “Random” button uses crypto.getRandomValues, which is cryptographically secure RNG. The source code is available on github.
